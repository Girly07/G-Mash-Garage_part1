G-Mash Garage Website - Part 1 (Building the Foundation)
---------------------------------------------------------
This is the initial version of the G-Mash Garage website project.

Structure:
- index.html (Homepage)
- about.html (About Us)
- services.html (Services)
- blog.html (Blog Page)
- contact.html (Contact Page)
- css/styles.css (Basic styling)

How to use:
1. Open index.html in a web browser to view the site.
2. All pages are linked together with navigation.
3. Edit the HTML and CSS files in a code editor such as Visual Studio Code.

This covers Part 1: Building the Foundation (HTML structure, navigation, file organisation, and basic styling).
